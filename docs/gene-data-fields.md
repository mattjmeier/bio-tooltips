# Gene Data Fields

Gene tooltips render selected fields from MyGene.info records.

| Area | Example fields |
| --- | --- |
| Identity | symbol, name, taxid |
| Summary | summary |
| Location | genomic position and chromosome |
| Pathways | KEGG, Reactome, WikiPathways |
| Protein context | InterPro domains, PDB structures |
| Literature | GeneRIFs |
| Links | NCBI Gene, Ensembl, UniProt, GeneCards, Wikipedia |

For exact type details, see the [generated MyGene types](./api/providers/mygene/types.md).
