# Gene Tooltips Overview

Gene tooltips use the MyGene.info adapter to resolve gene symbols and render rich biological context.

Common sections include summaries, species context, genomic location, pathways, protein domains, transcripts, structures, GeneRIFs, and external links.

```ts
import { GeneTooltip } from 'bio-tooltips/mygene';
import 'bio-tooltips/style.css';
```

Start with [Gene Usage](./gene-usage.md), then tune behavior in [Gene Configuration](./gene-configuration.md).
